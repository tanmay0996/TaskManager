# Task Manager API

Minimal Node + Express + MongoDB REST API for a task manager.

## Setup

1. Install dependencies (run in this folder):

   ```bash
   npm install
   ```

2. Create a `.env` file based on `.env.example`.

3. Run in development:

   ```bash
   npm run dev
   ```

4. Or run normally:

   ```bash
   npm start
   ```

> Note: Installs and commands are **not executed here**; run them locally in your environment.
